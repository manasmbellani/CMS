     ----------------------------------------
     |     STEPS TO TEST THE PROJECT        |
     ----------------------------------------      
     
     1-Install wamp, xampp or mamp to have a local server
     2-Copy the folder "CMS" to the workstation of the server that you have installed (in wamp C:/wamp/www, in xampp C:/xampp/htdocs, in mamp C:/mamp/htdocs)
     3-Enter in a browser the url "localhost" and enter in PhPMyAdmin
     4-Copy the structure of the database in the SQL tab (the structure is in the database folder)
     5-Enter the test data that are in the same folder, in the same way as with the structure
     6-Enter the different pages (localhost / CMS) (localhost / CMS / Administrator)
     7-User with privileges is pepito24 password 12345


     ----------------------------------------
     |     PASOS PARA PROBAR EL PROYECTO    |
     ----------------------------------------     
     
     1-Instalar wamp, xampp o mamp para tener un servidor local 
     2-Copiar la carpeta "CMS" en el sitio de trabajo del servido que hayas instalado (en wamp C:/wamp/www, en xampp C:/xampp/htdocs, en mamp C:/mamp/htdocs)
     3-Introducir en un navegador la url "localhost" y entrar en PhPMyAdmin
     4-Copiar la estructura de la base de datos en la pestaña SQL (la estructura se encuentra en la carpeta base de datos)
     5-Introducir los datos de prueba que se encuentran en la misma carpeta, de la misma manera que con la estructura
     6-Entrar a las diferentes paginas (localhost/CMS) (localhost/CMS/Administrador)
     7-El usuario con privilegios es pepito24 contraseña 12345
